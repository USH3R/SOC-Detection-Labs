## Summary
Completed the Junior Security Analyst room on TryHackMe. Learned the daily duties of a Level 1 SOC analyst, including monitoring alerts, investigating incidents, and understanding the roles of SOC team members.

## Key Takeaways
- SOC L1 analysts triage alerts and escalate complex cases to L2 analysts  
- Engineers configure security tools to support analysts  
- Managers oversee operations and coordinate responses  
- The Security Operations Center (SOC) is the central hub for defensive security
